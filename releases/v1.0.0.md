# v1.0.0

## What's new

- Calibration export + host report tool (`tools/calibration_report.py`)
- Opt-in adapters with host-side test coverage
- Single-header generation (`tools/amalgamate.py`) + CI validation
- CMake install/export support for downstream consumers (`find_package(loxbudget CONFIG REQUIRED)`)
- Expanded verification across GCC, Clang, sanitizers, fuzz smoke, cross builds and coverage

## Stability

Starting with v1.0.0, the public API declared in `include/loxbudget.h` follows semver stability expectations.

## Release artifacts

- `single_header/loxbudget.h`
- Static library + installed CMake package metadata
