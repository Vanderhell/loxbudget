# Handoff

This repository is structured for maintainable delivery rather than a one-off code drop.

## Verify before handoff

- `make test`
- `cmake -S . -B build && cmake --build build`
- `ctest --test-dir build --output-on-failure`
- If the ARM toolchain is available: `make cross-tiny`, `make cross-standard`, `make cross-full`

## Release inputs

- Update `CHANGELOG.md`
- Ensure release notes exist in `releases/<tag>.md`
- Push a `v*` tag to trigger `.github/workflows/release.yml`
- The release workflow publishes `single_header/loxbudget.h` as the primary packaged artifact

## Consumer-facing artifacts

- Static library target: `loxbudget`
- Installed public header: `include/loxbudget.h`
- CMake package export: `loxbudget::loxbudget`
- Single-header distribution generated by `python3 tools/amalgamate.py`

## Maintainer notes

- Public API stability starts at `v1.0.0`; breaking API changes should trigger a major version bump.
- Existing CI covers host GCC/Clang builds, sanitizers, fuzz smoke, adapter tests, cross-compilation, coverage and release artifact generation.
- Local scratch build directories are intentionally ignored via `.gitignore`.
